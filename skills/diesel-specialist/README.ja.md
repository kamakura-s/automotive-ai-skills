# Diesel Specialist(ディーゼル専門アシスタント)

[English](./README.md) | **日本語**

現代および旧世代のディーゼルエンジンのためのAIスキルです。コモンレールシステム、排ガス後処理装置(DPF/EGR/SCR)、グロープラグ、ターボチャージャー、ディーゼル特有の診断と整備をカバーします。

## 説明

ディーゼル専門アシスタントは、ガソリン車の経験では対応できない領域を扱います:高圧コモンレール燃料システム、DPF(ディーゼル微粒子捕集フィルター)の再生、アドブルー(尿素SCR)システム、そして圧縮着火エンジン特有の故障パターン。始動不良、排気煙、パワーダウン、排ガス系警告の診断を支援し、ディーゼルで高くつくことで有名な故障 — DPF詰まり、インジェクター摩耗、EGRによる吸気詰まり — を予防する整備習慣づくりを助けます。

**対象:** ディーゼル乗用車・トラック・バンのオーナー、ガソリン車から領域を広げる整備士、ディーゼル車を運用する事業者

## できること

- **燃料系** — コモンレールの原理と圧力、インジェクター不良の症状(ノック音、始動困難、煙、補正量の偏り)、フィードポンプとサプライポンプ(高圧ポンプ)の切り分け、燃料汚染への対応(水混入、誤給油、微生物汚染)、燃料フィルターの交換間隔とエア抜き手順
- **後処理装置** — DPFの仕組み、自然再生・強制再生・手動再生の違い、再生失敗と詰まりの原因(チョイ乗り、不適合オイル、上流の故障)、EGRの詰まり症状と清掃の選択肢、SCR/アドブルー:消費量の目安、結晶化、品質警告とカウントダウン制限
- **冷間始動** — グロープラグのテストと交換、吸気ヒーター、冬用軽油(凝固・曇点、流動性向上剤)、ディーゼルの圧縮に必要なバッテリーとクランキング速度
- **ターボチャージャー** — 可変ジオメトリー(VGT/VNT)の固着、ブーストリークの診断、オイル供給の問題、過給圧過大/不足のコード
- **診断** — 白煙/青煙/黒煙の読み方、燃費悪化、フェイルセーフ(リンプモード)のトリガー、ディーゼルランナウェイ(暴走)の知識と対処
- **整備** — ディーゼルではオイル規格がより重要(DPF付きはLow-SAPS/DL-1等)、タイミングベルト/チェーンの間隔、旧型エンジンのバルブ調整、クーラントとインジェクターシールの注意点
- **所有判断** — 自分の乗り方にディーゼルが合うか(チョイ乗りのDPFリスク)、中古ディーゼルの点検ポイント、多走行時の典型的なコストカーブ

## 推奨ワークフロー

1. **エンジンと排ガス装置を特定する。** 年式、メーカー、車種、わかればエンジン型式、DPF/EGR/SCR(アドブルー)の有無。
2. **乗り方を説明する。** チョイ乗りか高速中心かは、ディーゼル後処理装置の健康を左右する最大の要因です。
3. **警告を正確に報告する。** DPF・アドブルー・排ガス系の警告は段階的にエスカレートします — どのメッセージ・ランプか、カウントダウンの有無を正確に。
4. **可能ならライブデータを共有する。** DPFのスス堆積量、差圧、レール圧の値は診断を大幅に絞り込みます。
5. **手遅れになる前に動く。** DPFの堆積超過やアドブルーのカウントダウンは、DIYの選択肢が消えてディーラー対応が必須になる境界があります —「あとどれくらい猶予があるか」を聞いてください。
6. **再生またはドライブサイクルで修理を確認し**、該当の数値が正常域に戻ったことを確認します。

## 制限事項

- **排ガス装置の取り外し・無効化は扱いません。** DPF・EGR・アドブルーの除去は大半の国・地域で公道走行車には違法です。このスキルはこれらの装置を理解し、診断し、修理するためのものです。
- **高圧燃料系は危険です。** コモンレールの圧力は2,000barを超えることがあります — 稼働中のインジェクター配管は絶対に緩めないでください。噴射による傷害は重篤です。加圧部品を伴うテストは設備のあるプロの領域です。
- **強制再生やインジェクター補正値の登録は、多くの場合ディーラーレベルの診断機が必要**です — その境界に達したら明示します。
- **仕様の幅が大きい**(オイル規格、DPFの閾値、アドブルーの制御ロジック)ため、メーカー・仕向けごとに純正資料で確認してください。
- **旧式の機械式噴射ディーゼル**(列型・分配型ポンプ)は概念レベルでカバーしますが、ポンプのオーバーホールは専門業者の仕事です。
