# Car Audio & Electronics Assistant(カーオーディオ・電装アシスタント)

[English](./README.md) | **日本語**

社外オーディオと12V電装のためのAIスキルです。ヘッドユニット(カーナビ/ディスプレイオーディオ)交換、スピーカーとアンプの取り付け、ドライブレコーダーなどのアクセサリー配線、正しい電源・アース・ヒューズの取り方、そして現代のCANバス車が「ノー」と言う境界線を扱います。

## 説明

12V電装はDIYへの最も一般的な入口であり、同時にバッテリー上がり(暗電流)、アンプ破損、機嫌を損ねた車両電子系の最も一般的な発生源でもあります。このスキルはオーディオアップグレードの計画と実行(ヘッドユニット、スピーカー、アンプ、サブウーファー、デッドニング)、アクセサリーの正しい配線(ドラレコ、レーダー探知機、照明、インバーター)、そして「きれいな取り付け」と「火災の種」を分ける電源・アース・ヒューズの規律をカバーします。現代ならではの複雑さも明示します:CANバス車では、配線を自由に分岐できた時代は終わっています。

**対象:** 初めてのヘッドユニット交換からマルチアンプシステムの構築まで。ドラレコを電源直結したい人も。

## できること

- **ヘッドユニット** — サイズ(1DIN/2DIN/フローティング)、車種別ハーネスアダプターと「純正カプラーは絶対に切らない」理由、ステアリングリモコンインターフェース、アクセサリー電源の維持、アンテナ変換とバックカメラ変換、純正アンプのバイパス/統合
- **スピーカー** — セパレートとコアキシャル、能率とパワーマッチング、バッフルと奥行きの確認、ドアスピーカーにデッドニングが必要な理由、ハイレベル入力とRCA入力
- **アンプとサブウーファー** — 正しいゲイン設定(音量ツマミではありません)、電源線のゲージとバッテリーから30cm以内のヒューズ、アースポイントの品質、RCAと電源線を離して這わせる理由、エンクロージャー(シールドとポーテッド)の正直なトレードオフ、DSPの基礎
- **アクセサリー配線** — ドラレコと電源直結キット(正しい回路への「ヒューズ電源」:ACC連動か常時か)、駐車監視の消費電流計算と低電圧カットオフ、レーダー探知機、USB・インバーター負荷、LED化とハイフラ対策
- **電気の規律** — 消費電流の計算、ワイヤーゲージの選定、すべてのプラス線は電源側でヒューズ、アースは金属素地へ、大電流にはリレー、グラウンドループの回避(オルタネーターノイズの診断)
- **CANバスの理解** — 現代の車で正体不明の配線を針当て・分岐するとエラーやモジュール損傷につながる理由、インターフェースモジュールを使うべき場面、まだ安全に触れる回路、そして「ディーラーレベルの統合か、やらないか」が答えになる場合
- **トラブルシューティング** — オルタネーターノイズ、電源ON時のポップ音、取り付け後の暗電流の追跡、飛んだヒューズの原因究明
- **セキュリティ・利便装備** — 社外セキュリティの基礎、キーレス連動(**盗難防止装置の解除・バイパスは対象外**という明確な境界付き)

```

ヒューズの容量と位置、アースポイント、RCAの取り回し。
```

```text
ドラレコを電源直結したら、4日駐車でバッテリーが上がるようになりました。

```

## 推奨ワークフロー

1. **車両と電子装備のレベルを特定する。** 年式、メーカー、車種、グレード、純正アンプ/ディスプレイの有無 — これが変換アダプターの要否とCAN制約を最初に決めます。
2. **目的と予算を正直に伝える。**「普通の音量でクリアに」と「大音量の低音」は別のビルドです。目的に合わせて機材と配線をサイズします。
3. **電源計画が最初。** 合計消費電流、ワイヤーゲージ、ヒューズ、アースポイントを、部品の注文前に決めます。
4. **段階的にテストしながら取り付ける。** 最終組み付け前に仮接続で動作確認。各段階で警告灯が出ていないことを確認します。
5. **取り付け後:** 暗電流を測定し(モジュールのスリープ後、概ね50mA未満が目安)、ヒューズの容量と装着を再確認します。
6. **配線を記録する** — 未来の自分、次のオーナー、次のトラブルシューティングのために「どこから何を取ったか」を残してください。

## 制限事項

- **車種別の配線は必ず検証してください。** 配線色と回路の機能は仕向け・グレードで異なります。汎用の配線色一覧を信じず、テスターと車種別資料で確認を。
- **CANバス・モジュール統合には固い限界があります** — 純正ディスプレイ、デジタルアンプ、液晶メーターなどは専用インターフェース経由でしか統合できないか、そもそも不可能です。スキルはできない場合に「できない」と言います。
- **盗難防止装置は境界線です。** セキュリティの追加は扱いますが、イモビライザーのバイパス、アラームの解除、キーの複製は扱いません — この種の依頼は断ります。
- **エアバッグ(SRS)配線は不可侵です。** 黄色のコネクタ・ハーネスには触れません。その付近の作業(ダッシュ分解)はバッテリー切り離しと待機時間のルールに従います。
- **音質の判断は主観的で車室に依存します** — スキルはエンジニアリングを最適化します。最終的なチューニングはあなたの耳の仕事です。
- **大出力システムは充電系と相互作用します** — 大型アンプの搭載はオルタネーターとバッテリーの検討が必要な場合があり、スキルは指摘しますが、容量設計は専門店に。
