# Tire & Wheel Advisor(タイヤ&ホイールアドバイザー)

[English](./README.md) | **日本語**

タイヤとホイール専門のAIスキルです。サイズとフィッティングの計算、季節戦略(夏冬履き替え)、ロードインデックスと速度記号、TPMS(空気圧センサー)、摩耗診断、保管方法を扱います。

## 説明

タイヤは車の中で唯一路面に触れる部品であり、最も「買い方を間違えやすい」部品でもあります。タイヤ&ホイールアドバイザーは汎用スキルより一段深く踏み込みます:サイズ計算の完全版(外径、スピードメーター誤差、ロードインデックス)、夏冬の履き替え戦略、摩耗パターンを診断材料として読む方法、履き替え後のTPMSの挙動、そして宣伝カテゴリではなく実際の使い方に合わせたタイヤ選び。乗用車・SUV・バン・EVをカバーし、バイクは [Motorcycle Mechanic](../motorcycle-mechanic/README.ja.md) に委ねます。

**対象:** タイヤ・ホイールを買うすべての人、夏冬履き替えをする家庭、摩耗や振動の問題を調べているドライバー

## できること

- **サイズ計算** — サイズ表記の読み方(例:215/55R17 94V)、サイズ変更時の外径変化とスピードメーター/オドメーター誤差の計算、インチアップ・インチダウン(冬用ダウンサイズ)のルール、車両ラベル(指定空気圧ラベル)に対するロードインデックスと速度記号の要件
- **ホイールフィッティング** — ボルトパターン(PCD)、センターボア(ハブ径)とハブリング、オフセットがスクラブ半径とベアリング負荷に与える影響、ストラット/キャリパー/フェンダーとのクリアランス、ホイールの耐荷重、ホイールナットのトルクと増し締めの実践
- **選択** — 夏・オールシーズン・スタッドレスのコンパウンドの違いとそれぞれが本当に効く条件、SUV/トラックのハイウェイテレーン/オールテレーン/マッドテレーン、EV特有の考慮点(荷重、静粛性、低転がり抵抗、摩耗の速さ)、ランフラットのトレードオフ、ウェットグリップ・騒音のラベリング制度
- **季節戦略** — 履き替え時期の目安(7°Cルール)、2セット目のホイールが元を取れる条件、保管方法(置き方、温度、直射日光、年数の限界)
- **摩耗診断** — 摩耗パターンの読み方(センター摩耗、両肩摩耗、片減り、カッピング、フェザーエッジ)から空気圧・アライメント・サスペンションの問題を推定、残溝の測り方と製造年週(セリアル)の読み方、修理可能な損傷とバースト級の損傷の区別
- **TPMS** — 直接式と間接式、ローテーションや季節履き替え後の挙動、センサー電池とサービスキット、再学習(リラーン)手順のタイプ
- **実践** — ローテーションパターン(回転方向指定、前後異サイズ、フルサイズスペア)、積載・牽引時の空気圧戦略、新品タイヤの慣らし、混用のルール(良いペアはリアへ)

## プロンプト例

```text
純正サイズは215/55R17 94Vです。225/50R17は履けますか?外径と
スピードメーター誤差の計算を見せて、ロードインデックスと速度記号の
要件も確認してください。
```

```text
フロントタイヤだけ外側の肩が丸坊主で、リアは均等に減っています。
このパターンは何を意味していて、何を点検してもらうべきですか?
```

```text
雪国でEVに乗っています。2セット運用の戦略を提案してください:
サイズ、タイヤカテゴリ、冬用にインチダウンすべきかどうか。
```

```text
季節の履き替え後、TPMSランプが点いたままです。2セット目には
センサーが入っていません。安い順に選択肢を教えてください。
```

```text
タイヤ2本は溝たっぷりですが2019年30週製造の7年落ち、残り2本は
2年落ちです。交換、ローテーション、このまま — どうすべきですか?
```

## 推奨ワークフロー

1. **タイヤではなく車両ラベルから始める。** ドア開口部の指定空気圧ラベルのサイズ・ロードインデックス・速度記号・空気圧を共有してください — 今履いているタイヤがすでに間違っている可能性があります。
2. **実際の使い方を説明する。** 気候、年間走行距離、高速と市街地の比率、積載・牽引、最も重視する点(寿命、静粛性、グリップ、価格)。
3. **サイズ変更の場合:** 購入前にフル計算を求めてください — 外径の差、スピードメーター誤差、負荷能力の確認、クリアランスのリスク。
4. **摩耗の問題の場合:** 各タイヤの内側/中央/外側の残溝を測って数値で報告してください。位置ごとのパターンがそのまま診断になります。
5. **取り付けの詳細を確認する。** 規定トルクと50〜100km走行後の増し締め、TPMSの再学習、空気圧はラベル値に(サイドウォールの最大値ではなく)。
6. **製造年週とローテーション履歴を記録しておく**と、次回以降のセッションで経年と時期の助言ができます。

## 制限事項

- **車両ラベルとメーカー承認が正です。** 計算上は成立するサイズでも、車両として未承認だったり、地域の法規(ロードインデックス不足、フェンダーからのはみ出し等)に反する場合があります。日本でははみ出しと荷重が車検の確認点です。
- **タイヤの銘柄は常に入れ替わります** — カテゴリと選定基準には自信を持って助言しますが、現行モデルの具体名と価格は最新のテストと地域の在庫で確認してください。
- **タイヤの現物は見えません。** サイドウォールの損傷、パンク位置、コードの露出は物理的な点検が必要です。修理可否に迷ったらタイヤ店の判断が優先です。
- **振動の診断はサスペンションと重なります** — ホイールバランスは複数ある原因の1つです。消えない振動は [Vehicle Troubleshooting](../vehicle-troubleshooting/README.ja.md) へ。
- **冬用トラクションデバイス**(チェーン、スタッド)の規制は地域で異なります — 現地ルールを確認してください。
