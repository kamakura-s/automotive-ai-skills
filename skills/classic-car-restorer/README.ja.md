# Classic Car Restorer(旧車レストアアシスタント)

[English](./README.md) | **日本語**

クラシックカー・旧車のためのAIスキルです。ポイント点火、キャブレター、錆の診断、絶版部品の調達、そして「レストアか保存か」の判断を支援します。

## 説明

旧車レストアアシスタントは、OBDポート以前の技術と、現代車には存在しない判断を扱います:ポイントとコンデンサーの調整、キャブレターのオーバーホールとセッティング、錆が表面的か構造的かの見極め、何十年も前に生産終了した部品の探し方、そしてその車にふさわしいレストアのレベル — コンクール志向のフルレストア、現状を活かす保存(サンパシェティック・レストア)、レストモッド — の決め方。長期保管、現代の燃料との相性、50年前の車を現代の交通で安全に走らせる現実もカバーします。

**対象:** 初めて旧車を買う人、プロジェクト進行中のレストアラー、旧い車を維持し続けるオーナー

## できること

- **点火系** — ポイント/コンデンサーの理論、ギャップとドウェル角の調整、タイミングライトでの点火時期合わせ、コイルとバラスト抵抗のテスト、フルトラ化(電子点火化)の時期・方法とオリジナル性の代償
- **キャブレター** — 燃料計量の仕組み、オーバーホールの手順、プラグの焼け色によるジェッティングと混合気調整、マルチキャブの同調、エタノール混合燃料による問題と燃料系材質の対策
- **錆とボディ** — 表面錆・層状錆・貫通錆の区別、車検や安全に関わる構造部位(フレーム、サスペンション取付部、サイドシル)、購入前の錆診断、錆転換剤から切開溶接までの処置の選択肢
- **電装** — ダイナモ(直流発電機)とオルタネーターの違い、プラスアース車の注意、布巻き配線の危険、目立たない現代的保護の追加(リレー、ヒューズ、キルスイッチ)
- **機関・足回り** — ドラムブレーキの整備と調整、キングピンとグリスアップポイント、ノンシンクロミッションの操作、オーバーホール後の慣らし運転
- **部品調達** — NOS(新古品)・リプロ品・中古の使い分け、流用(インターチェンジ)の調べ方、リプロ品の品質の見極め、オーナーズクラブと専門店の活用、鋳造・機械加工によるワンオフ製作という最終手段
- **戦略** — レストアの優先順位と順序(機関を外装より先に)、正直な予算組み、オリジナル性と実用性のトレードオフ、来歴と価値を守る記録の残し方
- **維持** — 長期保管、保管後の再稼働、エタノール時代の燃料選び、クラシックカー保険(協定価額)や旧車登録制度の考え方

## 推奨ワークフロー

1. **車両と状態を特定する。** 年式、メーカー、車種、エンジン、オリジナル度、走行可否、不動期間。
2. **ゴールを決める。** コンクール仕上げ、実用車、現状保存、レストモッド — どれかで助言は根本的に変わります。
3. **バラす前に診断する。** まず状態評価チェックリストを求めてください。車は分解する方が組み立てるよりずっと速く進みます。
4. **正しい順序で進める。** 順序計画をもらってください:安全系と構造が最初、駆動系が次、外装・内装は最後。
5. **すべて記録する。** 分解前に写真、ボルト類は袋分けしてラベル、領収書は保管 — 記録習慣づくりも支援します。
6. **古い規定値は慎重に確認する。** 当時のマニュアルには誤植があり、規格も異なります(ウィットワース、プラスアース等)。クラブの知見と突き合わせてください。

## 制限事項

- **車種固有の知識には限界があります。** 生産時期による仕様変更、工場の癖、定番の対策は、オーナーズクラブや専門フォーラムに蓄積されています — アシスタントはそれを補完するもので、代替ではありません。
- **錆や金属の状態は遠隔では判断できません。** 説明と写真は助けになりますが、磁石・ピック・リフトが本当の道具です。
- **構造部の溶接、ブレーキ油圧系、ステアリングは安全に直結します** — プロに依頼するか点検を受けてください。旧車には現代の衝突安全装備が一切ありません。
- **価値とオリジナル性の判断は主観的で相場依存です** — 重要な車は、不可逆な変更の前に専門家や鑑定士に相談してください。
- **当時の規定値は廃止された単位・規格の場合があります**(ウィットワースねじ、ドウェル角度など)— 換算ミスは現実的なリスクです。二重に確認を。
