# Towing & Trailer Advisor(牽引・トレーラーアドバイザー)

[English](./README.md) | **日本語**

安全に牽引するためのAIスキルです。ヒッチの選定と定格、垂直荷重(ヒッチ荷重)と積載配分、トレーラーの配線とブレーキコントローラー、スウェイ(蛇行)の予防、トレーラーの整備を扱います。

## 説明

牽引は、小さなミスが高速道路の速度域で増幅される領域です:ヒッチクラスの不適合、垂直荷重の1割不足、整備されていないハブベアリング — どれか1つで旅行が終わるか、もっと悪い結果になります。牽引・トレーラーアドバイザーは、牽引車の能力、ヒッチとカプラーの選定、正しい積載、灯火とブレーキの配線、運転技術の違い、そして「必要なのにほぼ誰もやらない」トレーラーの整備スケジュールまで、チェーン全体をカバーします。運搬用トレーラー、ボートトレーラー、キャンピングトレーラー、車両運搬車に対応します。

**対象:** 初めて牽引する人、ボート・キャンピングトレーラーのオーナー、長期間放置したトレーラーを再稼働させる人

## できること

- **能力の計算** — 牽引能力の正しい読み方(車両総重量、連結総重量、ブレーキ付き/なしの限界、垂直荷重を含むペイロード)、チェーンの最弱リンクが限界を決める理由、実際の牽引での余裕の考え方
- **ヒッチとカプラー** — ヒッチクラスとレシーバーサイズ、ウェイトキャリング式とウェイトディストリビューション式(WDH)、WDHが必要になる条件、カプラーの調整とセーフティチェーンの交差掛け
- **積載** — 垂直荷重10〜15%ルールとその測り方、前方60/40の原則、トレーラー上の荷物固定、キャンピングトレーラーでの水と装備の配置、後ろ重心積載の帰結
- **配線とブレーキ** — 4ピン/7ピンの機能、灯火不良の診断(まずアース!)、電気ブレーキコントローラー(比例式とタイマー式、ゲイン設定の手順)、ブレーカウェイ装置とその電池、ボートトレーラーの慣性(サージ)ブレーキ
- **スウェイ(蛇行)** — スウェイの本当の原因(積載、速度、ジオメトリ、風)、予防の優先順位、スウェイ制御装置、そして発生時の正しい対処(ハンドル修正しない・ブレーキ踏まない — 手動トレーラーブレーキと一定スロットル)
- **運転技術** — 車間距離の延長、下り坂のギア規律、バックの基本(ハンドル下端に手)、内輪差・張り出しへの意識、ミラー要件
- **トレーラー整備** — ハブベアリングの整備(最も放置される項目。ボートトレーラーの水没後は特に)、トレーラータイヤの経年と残溝、ブレーキ点検、コネクタの腐食、フレームとカプラーの亀裂
- **法規の基礎** — トレーラー重量と免許区分の関係(日本では750kg超で原則けん引免許)、牽引時の速度制限、重量クラスごとのミラー・ブレーキ要件 — 一般論として提示し、詳細は地域の法規で確認

## 推奨ワークフロー

1. **数字から始める。** 牽引車の定格(カタログではなく取扱説明書から)、トレーラーの重量(可能なら実測)、ヒッチ/レシーバーの定格。
2. **用途を伝える。** トレーラーの種類、典型的な積載、距離、地形、頻度 — 毎週のボート降ろしと年1回の引っ越しでは助言が違います。
3. **セットアップの質問は順番に。** 能力 → ヒッチ → 積載 → 配線 → ブレーキコントローラーの順で。それぞれ前の項目に依存します。
4. **垂直荷重は推定せず測る**(体重計方式または専用スケール)。数値で報告してください。
5. **初回出発前:** 連結チェックリストと出発前10分の外周点検リストをもらい、毎回使ってください。
6. **保管後・水没後:** 積載する前に再稼働チェック(ベアリング、ブレーキ、灯火、タイヤ)を実施。

## 制限事項

- **定格と法的限界は車両・地域固有です。** スキルは計算と概念を教えます。拘束力のある数値は取扱説明書、車両ラベル、地域の交通当局が定めます(トレーラー重量と免許区分の関係は国によって大きく異なります。日本は総重量750kg超で原則けん引免許が必要です)。
- **実際の重量は検証できません。** 台貫(トラックスケール)での実測はどんな推定にも勝ります —「だいたい1トンくらい」が過積載の始まりです。
- **ブレーキコントローラーと配線は車両の電子系と相互作用します** — 純正牽引パッケージやCAN制御灯火を持つ新しい車では、汎用配線の近道が故障を招きます。迷ったら車種専用ハーネスを使ってください。
- **ヒッチの構造的な取り付け**(穴あけ、フレームマウント)はヒッチメーカーの車種別説明書とトルク指定に従ってください。
- **スウェイは走行安全に直結します** — スキルは予防と正しい対処を教えますが、どんな助言も「速度を落とし、正しく積む」ことの代わりにはなりません。
